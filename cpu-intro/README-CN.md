最重要的要理解的选项是 `PROCESS_LIST`（由 -l 或 --processlist 标志指定），它准确指定了每个运行的程序（或 '进程'）将要做什么。一个进程由指令组成，每个指令只能执行两件事之一：

* 使用 CPU
* 发出一个 I/O（并等待其完成）
